# ELASTIC-BEANSTALK
**AWS Elastic Beanstalk** is a Platform as a Service (PaaS) offered by Amazon Web Services that allows you to deploy and manage applications easily without managing infrastructure manually.

👉 You just upload your code
👉 AWS automatically handles:
    
    EC2 instances
    Load balancing
    Auto Scaling
    Monitoring
    Deployment
It is mainly used for deploying web applications.

**Supported Languages**

Elastic Beanstalk supports:

Java

Python

Node.js

PHP

.NET

Ruby

Docker
